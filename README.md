# GAMEJAM IFMS 2022
Este projeto foi desenvolvido para a GameJam do Instituto Federal de Educação, Ciência e Tecnologia de Mato Grosso do Sul realizado no ano de 2022 entre os dias 01/10 à 06/10.

## Versão da Engine
- Unity v2021.3.10;

## Plataforma de Lançamento:
- Windows 10/11 x64;
